# 🎉 Event Management Platform

> A modern full-stack Event Management Platform designed to simplify event creation, ticket booking, attendee management, and real-time event tracking.

Built with scalable web technologies and responsive UI principles to deliver a seamless experience for both organizers and attendees.

---

# 🚀 Features

## 👨‍💼 Organizer Features
- 📝 Create and manage events
- 🎟️ Manage ticket bookings
- 📊 Event analytics dashboard
- 📅 Schedule and update event details
- 📢 Send announcements & updates

## 🎫 User Features
- 🔍 Browse upcoming events
- 🎟️ Book tickets online
- 💳 Secure checkout workflow
- 📱 Responsive user interface
- ⭐ Save favorite events

## ⚡ Platform Features
- 🔐 Authentication & Authorization
- 🌐 REST API Integration
- 📊 Real-time event updates
- 🎨 Modern responsive UI
- ⚡ Fast and optimized performance

# ⚙️ Installation

## 📥 Clone the Repository

```bash
git clone https://github.com/your-username/event-management-platform.git
```

## 📂 Navigate to the Project

```bash
cd event-management-platform
```

## 📦 Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

# ▶️ Running the Project

## 🚀 Start Frontend

```bash
npm run dev
```

## ⚙️ Start Backend Server

```bash
npm start
```
